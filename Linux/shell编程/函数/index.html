е®ҡд№үеҮҪж•°

иҢғдҫӢпјҡ

жҹҘзңӢеҮҪж•°

еҲ йҷӨеҮҪж•°

еҮҪж•°и°ғз”Ё

еҮҪж•°зҡ„и°ғз”Ёж–№ејҸ

  • еҸҜеңЁдәӨдә’ејҸзҺҜеўғдёӢе®ҡд№үеҮҪж•°

  • еҸҜе°ҶеҮҪж•°ж”ҫеңЁи„ҡжң¬ж–Үд»¶дёӯдҪңдёәе®ғзҡ„дёҖйғЁеҲҶ

  • еҸҜж”ҫеңЁеҸӘеҢ…еҗ«еҮҪж•°зҡ„еҚ•зӢ¬ж–Үд»¶дёӯ

и°ғз”ЁпјҡеҮҪж•°еҸӘжңүиў«и°ғз”ЁжүҚдјҡжү§иЎҢпјҢйҖҡиҝҮз»ҷе®ҡеҮҪж•°еҗҚи°ғз”ЁеҮҪж•°пјҢеҮҪж•°еҗҚеҮәзҺ°зҡ„ең°ж–№пјҢдјҡиў«иҮӘеҠЁжӣҝжҚўдёәеҮҪж•°д»Јз Ғ

еҮҪж•°зҡ„з”ҹе‘Ҫе‘Ёжңҹпјҡиў«и°ғз”Ёж—¶еҲӣе»әпјҢиҝ”еӣһж—¶з»Ҳжӯў

дәӨдә’ејҸзҺҜеўғи°ғз”Ё

еңЁи„ҡжң¬дёӯе®ҡд№үеҸҠдҪҝз”Ё

еҮҪж•°еңЁдҪҝз”ЁеүҚеҝ…йЎ»е®ҡд№үпјҢеӣ жӯӨеә”е°ҶеҮҪж•°е®ҡд№үж”ҫеңЁи„ҡжң¬ејҖе§ӢйғЁеҲҶпјҢзӣҙиҮіshellйҰ–ж¬ЎеҸ‘зҺ°е®ғеҗҺжүҚиғҪдҪҝз”ЁпјҢи°ғз”ЁеҮҪж•°д»…дҪҝз”Ёе…¶еҮҪж•°еҗҚеҚіеҸҜ

иҢғдҫӢ

дҪҝз”ЁеҮҪж•°ж–Үд»¶

еҸҜд»Ҙе°Ҷз»ҸеёёдҪҝз”Ёзҡ„еҮҪж•°еӯҳе…ҘдёҖдёӘеҚ•зӢ¬зҡ„еҮҪж•°ж–Үд»¶пјҢз„¶еҗҺе°ҶеҮҪж•°ж–Үд»¶иҪҪе…ҘshellпјҢеҶҚиҝӣиЎҢи°ғз”ЁеҮҪж•°

еҮҪж•°ж–Үд»¶еҗҚеҸҜд»»ж„ҸйҖүеҸ–пјҢдҪҶжңҖеҘҪдёҺзӣёе…ід»»еҠЎжңүжҹҗз§ҚиҒ”зі»пјҢдҫӢеҰӮпјҡfunctions

дёҖж—ҰеҮҪж•°ж–Үд»¶иҪҪе…ҘshellпјҢе°ұеҸҜд»ҘеңЁе‘Ҫд»ӨиЎҢжҲ–и„ҡжң¬дёӯи°ғз”ЁеҮҪж•°гҖӮеҸҜд»ҘдҪҝз”Ёdelcare -f жҲ–set е‘Ҫд»ӨжҹҘзңӢжүҖжңүе®ҡд№үзҡ„еҮҪж•°пјҢе…¶иҫ“еҮәеҲ—иЎЁеҢ…жӢ¬е·Із»ҸиҪҪе…Ҙshellзҡ„жүҖжңүеҮҪж•°

иӢҘиҰҒж”№еҠЁеҮҪж•°пјҢйҰ–е…Ҳз”Ёunsetе‘Ҫд»Өд»ҺshellдёӯеҲ йҷӨеҮҪж•°гҖӮж”№еҠЁе®ҢжҜ•еҗҺпјҢеҶҚйҮҚж–°иҪҪе…ҘжӯӨж–Үд»¶

е®һзҺ°еҮҪж•°ж–Үд»¶зҡ„иҝҮзЁӢпјҡ

  • еҲӣе»әеҮҪж•°ж–Үд»¶пјҢеҸӘеӯҳж”ҫеҮҪж•°зҡ„е®ҡд№ү
  • еңЁshellи„ҡжң¬жҲ–дәӨдә’ејҸshellдёӯи°ғз”ЁеҮҪж•°ж–Үд»¶пјҢж јејҸеҰӮдёӢпјҡ

иҢғдҫӢ

еҮҪж•°иҝ”еӣһеҖј

еҮҪж•°зҡ„жү§иЎҢз»“жһңиҝ”еӣһеҖјпјҡ

  • дҪҝз”Ёechoзӯүе‘Ҫд»ӨиҝӣиЎҢиҫ“еҮә

  • еҮҪж•°дҪ“дёӯи°ғз”Ёе‘Ҫд»Өзҡ„иҫ“еҮәз»“жһң

еҮҪж•°зҡ„йҖҖеҮәзҠ¶жҖҒз Ғпјҡ

  • й»ҳи®ӨеҸ–еҶідәҺеҮҪж•°дёӯжү§иЎҢзҡ„жңҖеҗҺдёҖжқЎе‘Ҫд»Өзҡ„йҖҖеҮәзҠ¶жҖҒз Ғ

  • иҮӘе®ҡд№үйҖҖеҮәзҠ¶жҖҒз ҒпјҢе…¶ж јејҸдёәпјҡ

return д»ҺеҮҪж•°дёӯиҝ”еӣһпјҢз”ЁжңҖеҗҺзҠ¶жҖҒе‘Ҫд»ӨеҶіе®ҡиҝ”еӣһеҖј

return 0 ж— й”ҷиҜҜиҝ”еӣһ

return 1-255 жңүй”ҷиҜҜиҝ”еӣһ

жҹҘзңӢзҺҜеўғеҮҪж•°пјҡ

еҮҪж•°еҸӮж•°

еҮҪж•°еҸҜд»ҘжҺҘеҸ—еҸӮж•°пјҡ

  • дј йҖ’еҸӮж•°з»ҷеҮҪж•°пјҡеңЁеҮҪж•°еҗҚеҗҺйқўд»Ҙз©әзҷҪеҲҶйҡ”з»ҷе®ҡеҸӮж•°еҲ—иЎЁеҚіеҸҜпјҢеҰӮпјҡtestfunc arg1 arg2 вҖҰ

  • еңЁеҮҪж•°дҪ“дёӯеҪ“дёӯпјҢеҸҜдҪҝз”Ё1, 2, вҖҰи°ғз”ЁиҝҷдәӣеҸӮж•°пјӣиҝҳеҸҜд»ҘдҪҝз”Ё@, *, $#зӯүзү№ж®ҠеҸҳйҮҸ

иҢғдҫӢпјҡйҳ¶д№ҳ

иҢғдҫӢпјҡе®һзҺ°иҝӣеәҰжқЎеҠҹиғҪ

еҮҪж•°еҸҳйҮҸ

еҸҳйҮҸдҪңз”Ёеҹҹпјҡ

  • жҷ®йҖҡеҸҳйҮҸпјҡеҸӘеңЁеҪ“еүҚshellиҝӣзЁӢжңүж•ҲпјҢдёәжү§иЎҢи„ҡжң¬дјҡеҗҜеҠЁдё“з”ЁеӯҗshellиҝӣзЁӢпјӣеӣ жӯӨпјҢжң¬ең°еҸҳйҮҸзҡ„дҪңз”ЁиҢғеӣҙжҳҜеҪ“еүҚshellи„ҡжң¬зЁӢеәҸж–Үд»¶пјҢеҢ…жӢ¬и„ҡжң¬дёӯзҡ„еҮҪж•°

  • зҺҜеўғеҸҳйҮҸпјҡеҪ“еүҚshellе’Ңеӯҗshellжңүж•Ҳ

  • жң¬ең°еҸҳйҮҸпјҡеҮҪж•°зҡ„з”ҹе‘Ҫе‘ЁжңҹпјӣеҮҪж•°з»“жқҹж—¶еҸҳйҮҸиў«иҮӘеҠЁй”ҖжҜҒ

жіЁж„Ҹпјҡ

  • еҰӮжһңеҮҪж•°дёӯе®ҡд№үдәҶжҷ®йҖҡеҸҳйҮҸпјҢдё”еҗҚз§°е’ҢеұҖйғЁеҸҳйҮҸзӣёеҗҢпјҢеҲҷдҪҝз”Ёжң¬ең°еҸҳйҮҸ

  • з”ұдәҺжҷ®йҖҡеҸҳйҮҸе’ҢеұҖйғЁеҸҳйҮҸдјҡеҶІзӘҒпјҢе»әи®®еңЁеҮҪж•°дёӯеҸӘдҪҝз”Ёжң¬ең°еҸҳйҮҸ

еңЁеҮҪж•°дёӯе®ҡд№үжң¬ең°еҸҳйҮҸзҡ„ж–№жі•

еҮҪж•°йҖ’еҪ’

еҮҪж•°йҖ’еҪ’пјҡеҮҪж•°зӣҙжҺҘжҲ–й—ҙжҺҘи°ғз”ЁиҮӘиә«пјҢжіЁж„ҸйҖ’еҪ’еұӮж•°пјҢеҸҜиғҪдјҡйҷ·е…Ҙжӯ»еҫӘзҺҜ
