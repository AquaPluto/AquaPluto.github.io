е®үе…ЁжҠҖжңҜе’ҢйҳІзҒ«еўҷ

еҸ‘иЎЁдәҺ2025-08-21|жӣҙж–°дәҺ2025-08-22
|жҖ»еӯ—ж•°:2.6k|йҳ…иҜ»ж—¶й•ҝ:9еҲҶй’ҹ|жөҸи§ҲйҮҸ:

1 е®үе…ЁжҠҖжңҜе’ҢйҳІзҒ«еўҷ

1.1 е®үе…ЁжҠҖжңҜ

  • е…ҘдҫөжЈҖжөӢзі»з»ҹпјҲIntrusion Detection Systemsпјүпјҡзү№зӮ№жҳҜдёҚйҳ»ж–ӯд»»дҪ•зҪ‘з»ңи®ҝй—®пјҢйҮҸеҢ–гҖҒе®ҡдҪҚжқҘиҮӘеҶ…еӨ–зҪ‘з»ңзҡ„еЁҒиғҒжғ…еҶөпјҢдё»иҰҒд»ҘжҸҗдҫӣжҠҘиӯҰе’ҢдәӢеҗҺзӣ‘зқЈдёәдё»пјҢжҸҗдҫӣжңүй’ҲеҜ№жҖ§зҡ„жҢҮеҜјжҺӘж–Ҫе’Ңе®үе…ЁеҶізӯ–дҫқжҚ®,зұ»дјјдәҺзӣ‘жҺ§зі»з»ҹдёҖиҲ¬йҮҮз”Ёж—Ғи·ҜйғЁзҪІж–№ејҸ

  • е…ҘдҫөйҳІеҫЎзі»з»ҹпјҲIntrusion Prevention Systemпјүпјҡд»ҘйҖҸжҳҺжЁЎејҸе·ҘдҪңпјҢеҲҶжһҗж•°жҚ®еҢ…зҡ„еҶ…е®№еҰӮпјҡжәўеҮәж”»еҮ»гҖҒжӢ’з»қжңҚеҠЎж”»еҮ»гҖҒжңЁй©¬гҖҒи •иҷ«гҖҒзі»з»ҹжјҸжҙһзӯүиҝӣиЎҢеҮҶзЎ®зҡ„еҲҶжһҗеҲӨж–ӯпјҢеңЁеҲӨе®ҡдёәж”»еҮ»иЎҢдёәеҗҺз«ӢеҚідәҲд»Ҙйҳ»ж–ӯпјҢдё»еҠЁиҖҢжңүж•Ҳзҡ„дҝқжҠӨзҪ‘з»ңзҡ„е®үе…ЁпјҢдёҖиҲ¬йҮҮз”ЁеңЁзәҝйғЁзҪІж–№ејҸ

  • йҳІзҒ«еўҷпјҲ FireWall пјүпјҡйҡ”зҰ»еҠҹиғҪпјҢе·ҘдҪңеңЁзҪ‘з»ңжҲ–дё»жңәиҫ№зјҳпјҢеҜ№иҝӣеҮәзҪ‘з»ңжҲ–дё»жңәзҡ„ж•°жҚ®еҢ…еҹәдәҺдёҖе®ҡзҡ„и§„еҲҷжЈҖжҹҘпјҢе№¶еңЁеҢ№й…Қжҹҗи§„еҲҷж—¶з”ұи§„еҲҷе®ҡд№үзҡ„иЎҢдёәиҝӣиЎҢеӨ„зҗҶзҡ„дёҖз»„еҠҹиғҪзҡ„з»„д»¶пјҢеҹәжң¬дёҠзҡ„е®һзҺ°йғҪжҳҜй»ҳи®Өжғ…еҶөдёӢе…ій—ӯжүҖжңүзҡ„йҖҡиҝҮеһӢи®ҝй—®пјҢеҸӘејҖж”ҫе…Ғи®ёи®ҝй—®зҡ„зӯ–з•Ҙ,дјҡе°ҶеёҢжңӣеӨ–зҪ‘и®ҝй—®зҡ„дё»жңәж”ҫеңЁDMZ(demilitarized zone)зҪ‘з»ңдёӯ

  • йҳІж°ҙеўҷпјҲWaterwallпјүпјҡдёҺйҳІзҒ«еўҷзӣёеҜ№пјҢйҳІж°ҙеўҷжҳҜдёҖз§ҚйҳІжӯўеҶ…йғЁдҝЎжҒҜжі„жјҸзҡ„е®үе…Ёдә§е“ҒгҖӮе®ғеҲ©з”ЁйҖҸжҳҺеҠ и§ЈеҜҶпјҢиә«д»Ҫи®ӨиҜҒпјҢи®ҝй—®жҺ§еҲ¶е’Ңе®Ўи®Ўи·ҹиёӘзӯүжҠҖжңҜжүӢж®өпјҢеҜ№ж¶үеҜҶдҝЎжҒҜпјҢйҮҚиҰҒдёҡеҠЎж•°жҚ®е’ҢжҠҖжңҜдё“еҲ©зӯүж•Ҹж„ҹдҝЎжҒҜзҡ„еӯҳеӮЁпјҢдј ж’ӯе’ҢеӨ„зҗҶиҝҮзЁӢпјҢе®һж–Ҫе®үе…ЁдҝқжҠӨпјӣжңҖеӨ§йҷҗеәҰең°йҳІжӯўж•Ҹж„ҹдҝЎжҒҜжі„жјҸгҖҒиў«з ҙеқҸе’Ңиҝқи§„еӨ–дј пјҢе№¶е®Ңж•ҙи®°еҪ•ж¶үеҸҠж•Ҹж„ҹдҝЎжҒҜзҡ„ж“ҚдҪңж—Ҙеҝ—пјҢд»Ҙдҫҝж—ҘеҗҺе®Ўи®ЎгҖӮ

1.2 йҳІзҒ«еўҷзҡ„еҲҶзұ»

жҢүдҝқжҠӨиҢғеӣҙеҲ’еҲҶпјҡ

  • дё»жңәйҳІзҒ«еўҷпјҡжңҚеҠЎиҢғеӣҙдёәеҪ“еүҚдёҖеҸ°дё»жңә

  • зҪ‘з»ңйҳІзҒ«еўҷпјҡжңҚеҠЎиҢғеӣҙдёәйҳІзҒ«еўҷдёҖдҫ§зҡ„еұҖеҹҹзҪ‘

жҢүе®һзҺ°ж–№ејҸеҲ’еҲҶ:

  • зЎ¬д»¶йҳІзҒ«еўҷпјҡеңЁдё“з”ЁзЎ¬д»¶зә§еҲ«е®һзҺ°йғЁеҲҶеҠҹиғҪзҡ„йҳІзҒ«еўҷпјӣеҸҰдёҖдёӘйғЁеҲҶеҠҹиғҪеҹәдәҺиҪҜд»¶е®һзҺ°

  • иҪҜд»¶йҳІзҒ«еўҷпјҡиҝҗиЎҢдәҺйҖҡз”ЁзЎ¬д»¶е№іеҸ°д№ӢдёҠзҡ„йҳІзҒ«еўҷзҡ„еә”з”ЁиҪҜд»¶пјҢWindows йҳІзҒ«еўҷ ISA вҖ“> ForefrontTMG

жҢүзҪ‘з»ңеҚҸи®®еҲ’еҲҶпјҡ

  • зҪ‘з»ңеұӮйҳІзҒ«еўҷпјҡOSIжЁЎеһӢдёӢеӣӣеұӮпјҢеҸҲз§°дёәеҢ…иҝҮж»ӨйҳІзҒ«еўҷ

  • еә”з”ЁеұӮйҳІзҒ«еўҷ/д»ЈзҗҶжңҚеҠЎеҷЁпјҡproxy д»ЈзҗҶзҪ‘е…іпјҢOSIжЁЎеһӢдёғеұӮ

еҢ…иҝҮж»ӨйҳІзҒ«еўҷ

зҪ‘з»ңеұӮеҜ№ж•°жҚ®еҢ…иҝӣиЎҢйҖүжӢ©пјҢйҖүжӢ©зҡ„дҫқжҚ®жҳҜзі»з»ҹеҶ…и®ҫзҪ®зҡ„иҝҮж»ӨйҖ»иҫ‘пјҢиў«з§°дёәи®ҝй—®жҺ§еҲ¶еҲ—иЎЁпјҲACLпјүпјҢйҖҡиҝҮжЈҖжҹҘж•°жҚ®жөҒдёӯжҜҸдёӘж•°жҚ®зҡ„жәҗең°еқҖпјҢзӣ®зҡ„ең°еқҖпјҢжүҖз”Ёз«ҜеҸЈеҸ·е’ҢеҚҸи®®зҠ¶жҖҒзӯүеӣ зҙ пјҢжҲ–д»–д»¬зҡ„з»„еҗҲжқҘзЎ®е®ҡжҳҜеҗҰе…Ғи®ёиҜҘж•°жҚ®еҢ…йҖҡиҝҮ
дјҳзӮ№пјҡеҜ№з”ЁжҲ·жқҘиҜҙйҖҸжҳҺпјҢеӨ„зҗҶйҖҹеәҰеҝ«дё”жҳ“дәҺз»ҙжҠӨ
зјәзӮ№пјҡж— жі•жЈҖжҹҘеә”з”ЁеұӮж•°жҚ®пјҢеҰӮз—…жҜ’зӯү

еә”з”ЁеұӮйҳІзҒ«еўҷ

еә”з”ЁеұӮйҳІзҒ«еўҷ/д»ЈзҗҶжңҚеҠЎеһӢйҳІзҒ«еўҷпјҢд№ҹз§°дёәд»ЈзҗҶжңҚеҠЎеҷЁпјҲProxy Server)е°ҶжүҖжңүи·Ёи¶ҠйҳІзҒ«еўҷзҡ„зҪ‘з»ңйҖҡдҝЎй“ҫи·ҜеҲҶдёәдёӨж®өеҶ…еӨ–зҪ‘з”ЁжҲ·зҡ„и®ҝй—®йғҪжҳҜйҖҡиҝҮд»ЈзҗҶжңҚеҠЎеҷЁдёҠзҡ„вҖңй“ҫжҺҘвҖқжқҘе®һзҺ°
дјҳзӮ№пјҡеңЁеә”з”ЁеұӮеҜ№ж•°жҚ®иҝӣиЎҢжЈҖжҹҘпјҢжҜ”иҫғе®үе…Ё
зјәзӮ№пјҡеўһеҠ йҳІзҒ«еўҷзҡ„иҙҹиҪҪ

жҸҗзӨәпјҡзҺ°е®һз”ҹдә§зҺҜеўғдёӯжүҖдҪҝз”Ёзҡ„йҳІзҒ«еўҷдёҖиҲ¬йғҪжҳҜдәҢиҖ…з»“еҗҲдҪ“пјҢеҚіе…ҲжЈҖжҹҘзҪ‘з»ңж•°жҚ®пјҢйҖҡиҝҮд№ӢеҗҺеҶҚйҖҒеҲ°еә”з”ЁеұӮеҺ»жЈҖжҹҘ

1.3 йҳІзҒ«еўҷзҡ„дҪңз”Ё

йҳІзҒ«еўҷзҡ„еҹәжң¬еҠҹиғҪжҳҜзӣ‘жҺ§е’ҢжҺ§еҲ¶иҝӣеҮәзҪ‘з»ңзҡ„ж•°жҚ®жөҒгҖӮе®ғж №жҚ®йў„е…Ҳе®ҡд№үзҡ„е®үе…Ёи§„еҲҷжқҘе®ЎжҹҘжҜҸдёҖдёӘж•°жҚ®еҢ…пјҢиҝҷдәӣи§„еҲҷеҸҜиғҪеҹәдәҺIPең°еқҖгҖҒз«ҜеҸЈеҸ·гҖҒеҚҸи®®зұ»еһӢз”ҡиҮіжҳҜе…·дҪ“зҡ„еҶ…е®№гҖӮд»»дҪ•дёҚз¬ҰеҗҲи§„еҲҷзҡ„ж•°жҚ®еҢ…йғҪдјҡиў«йҳІзҒ«еўҷжӢҰжҲӘпјҢд»ҺиҖҢйҳІжӯўжҪңеңЁзҡ„ж”»еҮ»е’ҢжңӘз»ҸжҺҲжқғзҡ„и®ҝй—®гҖӮ

йҷӨдәҶеҹәжң¬зҡ„еҢ…иҝҮж»ӨеҠҹиғҪпјҢзҺ°д»ЈйҳІзҒ«еўҷиҝҳе…·еӨҮжӣҙй«ҳзә§зҡ„е®үе…Ёзү№жҖ§пјҢеҰӮе…ҘдҫөжЈҖжөӢзі»з»ҹпјҲIDSпјүгҖҒе…ҘдҫөйҳІеҫЎзі»з»ҹпјҲIPSпјүе’ҢиҷҡжӢҹдё“з”ЁзҪ‘з»ңпјҲVPNпјүзӯүгҖӮиҝҷдәӣеҠҹиғҪдҪҝеҫ—йҳІзҒ«еўҷжҲҗдёәдәҶзҪ‘з»ңе®үе…Ёзҡ„з¬¬дёҖйҒ“йҳІзәҝгҖӮ

2 LinuxйҳІзҒ«еўҷзҡ„еҹәжң¬и®ӨиҜҶ

2.1 Netfilter

LinuxйҳІзҒ«еўҷжҳҜз”ұNetfilterз»„д»¶жҸҗдҫӣзҡ„пјҢNetfilterе·ҘдҪңеңЁеҶ…ж ёз©әй—ҙпјҢйӣҶжҲҗеңЁlinuxеҶ…ж ёдёӯ

Netfilter жҳҜLinux 2.4.xд№ӢеҗҺж–°дёҖд»Јзҡ„LinuxйҳІзҒ«еўҷжңәеҲ¶пјҢжҳҜlinuxеҶ…ж ёзҡ„дёҖдёӘеӯҗзі»з»ҹгҖӮNetfilterйҮҮз”ЁжЁЎеқ—еҢ–и®ҫи®ЎпјҢе…·жңүиүҜеҘҪзҡ„еҸҜжү©е……жҖ§пјҢжҸҗдҫӣжү©еұ•еҗ„з§ҚзҪ‘з»ңжңҚеҠЎзҡ„з»“жһ„еҢ–еә•еұӮжЎҶжһ¶гҖӮNetfilterдёҺIPеҚҸи®®ж ҲжҳҜж— зјқеҘ‘еҗҲпјҢе№¶е…Ғи®ёеҜ№ж•°жҚ®жҠҘиҝӣиЎҢиҝҮж»ӨгҖҒең°еқҖиҪ¬жҚўгҖҒеӨ„зҗҶзӯүж“ҚдҪң

2.3 netfilter дёӯдә”дёӘеӢҫеӯҗеҮҪж•°е’ҢжҠҘж–ҮжөҒеҗ‘

NetfilterеңЁеҶ…ж ёдёӯйҖүеҸ–дә”дёӘдҪҚзҪ®ж”ҫдәҶдә”дёӘhook(еӢҫеӯҗ) function(INPUTгҖҒOUTPUTгҖҒFORWARDгҖҒPREROUTINGгҖҒPOSTROUTING)пјҢиҖҢиҝҷдә”дёӘhook functionеҗ‘з”ЁжҲ·ејҖж”ҫпјҢз”ЁжҲ·еҸҜд»ҘйҖҡиҝҮдёҖдёӘе‘Ҫд»Өе·Ҙе…·пјҲiptablesпјүеҗ‘е…¶еҶҷе…Ҙи§„еҲҷпјҢеҶ…ж ёдёӯзҡ„еӢҫеӯҗеҮҪж•°ж №жҚ®йў„и®ҫзҡ„и§„еҲҷиҝӣиЎҢе·ҘдҪңпјҢиҫҫеҲ°еҜ№жөҒз»Ҹзҡ„ж•°жҚ®еҢ…иҝӣиЎҢиҝҮж»ӨпјҢжӢ’з»қпјҢиҪ¬еҸ‘зӯүеҠҹиғҪгҖӮ

з”ұдҝЎжҒҜиҝҮж»ӨиЎЁпјҲtableпјүз»„жҲҗпјҢеҢ…еҗ«жҺ§еҲ¶IPеҢ…еӨ„зҗҶзҡ„и§„еҲҷйӣҶпјҲrulesпјүпјҢи§„еҲҷиў«еҲҶз»„ж”ҫеңЁй“ҫпјҲchainпјүдёҠ

зү№еҲ«иҜҙжҳҺ

д»Һ Linux kernel 4.2 зүҲд»ҘеҗҺпјҢnetfilter еңЁprerouting еүҚеҠ дәҶдёҖдёӘ ingress еӢҫеӯҗеҮҪж•°гҖӮеҸҜд»ҘдҪҝз”ЁиҝҷдёӘж–°зҡ„е…ҘеҸЈжҢӮй’©жқҘиҝҮж»ӨжқҘиҮӘз¬¬2еұӮзҡ„жөҒйҮҸпјҢиҝҷдёӘж–°жҢӮй’©жҜ”йў„и·Ҝз”ұиҰҒж—©пјҢеҹәжң¬дёҠжҳҜ tc е‘Ҫд»ӨпјҲжөҒйҮҸжҺ§еҲ¶е·Ҙе…·пјүзҡ„жӣҝд»Је“ҒгҖӮ

дёүз§ҚжҠҘж–ҮжөҒеҗ‘

  • жөҒе…Ҙжң¬жңәпјҡPREROUTING вҖ“> INPUTвҖ“>з”ЁжҲ·з©әй—ҙиҝӣзЁӢ

  • жөҒеҮәжң¬жңәпјҡз”ЁжҲ·з©әй—ҙиҝӣзЁӢ вҖ“>OUTPUTвҖ“> POSTROUTING

  • иҪ¬еҸ‘пјҡPREROUTING вҖ“> FORWARD вҖ“> POSTROUTING

INPUT й“ҫ

  • иҜҘй“ҫз”ЁдәҺеӨ„зҗҶиҝӣе…Ҙжң¬ең°зі»з»ҹзҡ„жөҒйҮҸпјҲеҚізӣ®ж ҮжҳҜжң¬жңәзҡ„жөҒйҮҸпјүгҖӮ
  • жүҖжңүеҲ°иҫҫжң¬жңәзҡ„жөҒйҮҸйғҪдјҡз»ҸиҝҮ INPUT й“ҫиҝӣиЎҢжЈҖжҹҘ йҖӮз”ЁдәҺдҫӢеҰӮ SSHгҖҒHTTP зӯүжңҚеҠЎзҡ„иҝһжҺҘиҜ·жұӮ

OUTPUT й“ҫ

  • иҜҘй“ҫз”ЁдәҺеӨ„зҗҶд»Һжң¬ең°зі»з»ҹеҸ‘еҮәзҡ„жөҒйҮҸпјҲеҚіжәҗжҳҜжң¬жңәзҡ„жөҒйҮҸпјүгҖӮ
  • жүҖжңүз”ұжң¬жңәеҸ‘еҮәзҡ„жөҒйҮҸйғҪдјҡз»ҸиҝҮ OUTPUT й“ҫ йҖӮз”ЁдәҺдҫӢеҰӮд»Һжң¬жңәи®ҝй—®еӨ–йғЁзҪ‘з«ҷзҡ„иҜ·жұӮ

FORWARD й“ҫ

  • иҜҘй“ҫз”ЁдәҺеӨ„зҗҶиў«и·Ҝз”ұйҖҡиҝҮжң¬жңәзҡ„жөҒйҮҸпјҲеҚіжөҒйҮҸдёҚдјҡеҲ°иҫҫжң¬жңәпјҢиҖҢжҳҜиҪ¬еҸ‘еҲ°е…¶д»–зҪ‘з»ңпјүгҖӮ
  • жӯӨй“ҫд»…йҖӮз”ЁдәҺеҒҡи·Ҝз”ұзҡ„жңәеҷЁ йҖӮз”ЁдәҺдҫӢеҰӮи·Ҝз”ұеҷЁиҪ¬еҸ‘зҡ„жөҒйҮҸ

PREROUTING й“ҫ

  • иҜҘй“ҫз”ЁдәҺеңЁи·Ҝз”ұеҶізӯ–д№ӢеүҚеӨ„зҗҶжөҒйҮҸгҖӮ
  • йҖҡеёёз”ЁдәҺзӣ®ж Үең°еқҖиҪ¬жҚўпјҲDNATпјүпјҢеңЁж•°жҚ®еҢ…еҲ°иҫҫи·Ҝз”ұеүҚдҝ®ж”№е…¶зӣ®ж Үең°еқҖ
  • йҖӮз”ЁдәҺдҫӢеҰӮзҪ‘з»ңең°еқҖиҪ¬жҚўпјҲNATпјүж—¶пјҢдҝ®ж”№зӣ®ж Үең°еқҖзҡ„и§„еҲҷ

POSTROUTING й“ҫ

  • иҜҘй“ҫз”ЁдәҺеңЁи·Ҝз”ұеҶізӯ–д№ӢеҗҺеӨ„зҗҶжөҒйҮҸгҖӮ
  • йҖҡеёёз”ЁдәҺжәҗең°еқҖиҪ¬жҚўпјҲSNATпјүпјҢеңЁж•°жҚ®еҢ…зҰ»ејҖжң¬жңәеүҚдҝ®ж”№жәҗең°еқҖ
  • дҫӢеҰӮ NAT дёӯдҝ®ж”№жәҗең°еқҖзҡ„и§„еҲҷ
